## Proyecto Caso de Uso

La aplicacion solo contiene un buscado de noticias y el detalle de las mismas

- El proyeto contiene las migraciones necesarias para funcionar.
- El proyecto contiene seeder iniciales .
- Para ingresar los seeder y las migracionde recuerde usar en la consola del proyecto (php artisan migrate --seed).
- Contiene un login que funciona pero no existen modulos que esten protegidos.
